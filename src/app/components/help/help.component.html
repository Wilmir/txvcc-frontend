Step 1: Create A New Network

In your networks page, click the card with a plus ('+') icon. (Alternatively, you can also click the plus icon in the nav bar).

You will be redirected to a form asking for the name of your network.

  • On the Name field, input the name of the network you are creating. It is suggested that you use a unique name, that will enable you to easily identify this network as the number of networks that you app-create-network in this application increase.
  • On the Description field, provide a descriptive characteristic of your network.
  • When you Submit, the network template will be created and a new card will be created in your networks page. The submission will automatically bring you to the networks page.

Step 2: Add The List of Nodes

To start adding your nodes or to update your network, click on the network's card from the network page.

If your network is new, you will see an empty network page. Otherwise, you will see the diagram and other network details.

The sidebar on the left will have a file upload button which you can use to upload the excel file (.XLSX) of your nodes.

 Nodes Excel File Requirement

  • The excel file should contain 2 columns: name and homing specifier of the node.
  • To identify a homing node, the value of homing column should be set to 1, if it is a regular node set it to 0.
  • You can also specify more than one homing node in the list

 Sample:

name homing
Dublin 1
Swords 0
Tallagt 0
Dun Laoighre 0
Wicklow 0
Naas 0
Drogheda 0
Navan 0

 Uploading Your File

When your excel file is ready, click 'Choose file' and the click Upload. The system will look for the 'name' and 'type' column and add those data to the network. Once they are uploaded, the network canvass will refresh and reflect the nodes.

  • Green nodes are the homing nodes
  • Yellow nodes are the regular nodes

You will also find a downloadable table containing the list of your nodes assigned with and IDs. You will need these IDs when uploading links and services.

Step 2: Add The List of Links

The Upload Link button will appear after the list of nodes have been uploaded.

As a requirement, you can only add links to a network, when the nodes for these links are already uploaded. If you have not uploaded the nodes yet, please see Step 1: Add List ofNodes

 Links Excel File Requirement

  • The excel file should contain 4 columns: source_id, target_id, capacity, type
  • The source_id, and target_id can be obtained by downloading the nodes list below the network diagram
  • The capacity unit is suggested to be in Mbps. You can apply other unit as well as long as it is the same as the unit that you will use with the services.

 Sample:

 Uploading Your File

When your excel file is ready, click 'Choose file' and the click Upload. The system will look for the 'source_id', 'target_id', 'capacity', 'type' columns and add those data to the network. Once they are uploaded, the network canvass will refresh and reflect the links. You can skip the Add Services Upload or upload both services and links at the same time.

Step 2: Add The List of Services

The Upload Services button will appear after the list of nodes have been uploaded.

As a requirement, you can only add services to a network, when the nodes for these services are already uploaded. If you have not uploaded the nodes yet, please see Step 1: Add List ofNodes

 Services Excel File Requirement

The excel file should contain 4 columns: type, capacity, node_id, homing_node_id

  • The type can be any service provided by the node
  • The capacity unit is suggested to be in Mbps. You can apply other unit as well as long as it is the same as the unit that you will use with the services.
  • The node_id is the ID of the node where this service is served. The ID should correspond to the ID provided by TxVCC.
  • The homing_node_id is the ID of the node where this service is homing to. The ID should correspond to the ID provided by TxVCC.

 Sample:

Ensure that all the source_id and target_id of the nodes in your excel file are the IDs of the nodes that are already uploaded in the network. If at least, one is not present in the network, the uploaded links will neither be accepted nor saved.

3G 50 1146 1148
WiFi 40 1147 1148
LTE 1000 1148 1148

 Uploading Your File

When your excel file is ready, click 'Choose file' and the click Upload. The system will look for the 'type', 'capacity', 'node_id', 'homing_node_id' columns and add those data to the network. Once they are uploaded, the network canvass will refresh and reflect the services.

  • Hovering your mouse over any node will show the statistics of each node
  • The Network Summary will provide the high level summary of the status of each link
  • Each link will update its color: Red >= 90%, Orange >=50%, Greed <50% Utilization

You will also find a downloadable table containing the list of network links assigned with and IDs underneath the network diagram
