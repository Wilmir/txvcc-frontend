Network Graph Visualization. Connecting the dots

TxVCC provides automatic visualization of different network topologies: ring, tree, bus, star or their combinations. It uses the force-directed graph layout in the D3.JS library to create the network graph simulations.

Network Capacity Utilization. Understand your network

The application runs a customized algorithm which assumes that the best path between the services' node and their homing node is th shortest path. The path that requires the minimum number of transmission link infrastructure.

Data Protection. Usage, Security and Confidentiality.

The user can download the result of the algoritmic computation to propose changes or improvements in the network. Only the creators of the network can view, update, download or delete their network's data.

Each user has their password encrypted when they register. Upon login, each user is given a token which allows them to access their data. This token is cleared out from the user's computer when they log out.
